## Como rodar
A todos os arquivos `.sh` devem ser dadas permissão de execução:

    ls *.sh | xargs chmod +x 
    ./srun.sh <RANDOM_SEED> <X_INICIAL> <Y_INICIAL> <QTD_TURNOS>

Por exemplo:

    ./sbuild.sh
    ./srun.sh 777 5 5 8
